# Реализация простого TCP сервера

### Описание
Простейшая реализация TCP сервера и клиента с большим колличеством коментариев и понятным кодом. Клиент отправляет серверу запрос 
* quit - прервать соединение
* add - добавление координат точек, клиен должен отправиь 4 координаты 2х точек: центр окружности и точка, лежащая на окружности
* sq - узнать площадь круга по последним 2м добавленным точкам
и сервер возвращает результат 
Одновременно могжет быть несколько клиентов

### Запуск

Открыть 2 консоли
    1) В первой запустить `go run proto.go server.go` 
    2) Во второй `go run proto.go client.go`
Если необходимо несколько клиентов, то каждый в отдельной консоли аналогично п.2)
